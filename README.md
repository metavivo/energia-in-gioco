# Energia in Gioco

Gioco didattico web per imparare la fisica dell'energia meccanica.

## Contenuto

- Simulazione di un carrello su rampa
- Controlli per massa, altezza, velocita iniziale e attrito
- Calcolo di energia potenziale, cinetica e meccanica
- Quiz rapido con punteggio e feedback

## Avvio

Apri `index.html` in un browser.

## Pubblicazione su GitHub Pages

Il progetto include il workflow `.github/workflows/deploy-pages.yml`.

Dopo aver caricato il repository su GitHub:

1. apri le impostazioni del repository
2. vai in `Settings > Pages`
3. in `Source` seleziona `GitHub Actions`
4. fai push sul branch `main`

GitHub pubblichera il sito con un link del tipo:

`https://metavivo.github.io/energia-in-gioco/`
